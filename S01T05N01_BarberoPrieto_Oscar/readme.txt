Procés per executar Java des de consola:

Per poder executar una aplicació Java des de consola hem de seguir els següents passos:

1. Crear un fitxer amb el mateix nom que la classe.

2. Accedir al fitxer mitjançant la seva ruta.

		cd ruta del fitxer

2. Compilar el programa.

		javac ruta del fitxer

3. Es crea un fitxer amb extensió .class. Un creat ja podem executar el programa.

		java ruta del fitxer (no cal posar l'extensió .java)
		
Exemples:

	->		cd  /S01_T05-IT_ACADEMY/S01T05N01_BarberoPrieto_Oscar/src

	->		javac Exercici1/LlistaArxiusAlfabeticament.java
	->		java Exercici1.LlistaArxiusAlfabeticament

	->		javac Exercici2/LlistaArxiusAlfabeticament.java
	->		java Exercici2.LlistaArxiusAlfabeticament

	->		javac Exercici3/LlistaDirectoris.java
	->		java Exercici3.LlistaDirectoris

	->		javac Exercici4/LlegirText.java
	->		java Exercici4.LlegirText

	->		javac Exercici5/ObjecteSerial.java
	->		java Exercici5.ObjecteSerial
